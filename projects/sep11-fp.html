Sep11 Freedom Project

The "Freedom Project" for SEP11 is all about creating anything with javascript and to connect it to something you're passionate about. We work on it during the entire school year to show off what we've learned and express our creativity.

For my project, I created a website where you can find your lost pets with an additional feature of Javascript instead of just HTML, CSS, and Github. I chose to create a website because I was familiar with it and I like advocation through creativity and not just verbally.

In an addition to Javascipt, I learned a tool related with Javascript called Firebase. Firebase uses Javascript to store information and build an authentication system. I had a rocky start and it was hard to build a storage system to save information. A takeaway I had to solve this challenge was to ask peers for help and to watch Net Ninja videos to try to replicate code in a way work for my website. My next step would be trying to be able to incorporate a way to save images as well instead of just text.
